Solutions to [Advent of Code](https://adventofcode.com/) puzzles written in TypeScript.

**📥 Install dependencies using *`npm`*:**

```bash
npm i
```

**🏃 Run a solution using `ts-node`:**

```bash
# 2021 day 1 puzzle
ts-node src/2021/day01
```

**ℹ️ ️`node v14+` is required.**
